# ThinkML
**Asistente para definir el enfoque de modelado**

ThinkML es una herramienta interactiva que ayuda a identificar el enfoque correcto de ciencia de datos antes de escribir código.

## Por qué existe
Muchos proyectos de machine learning fallan no por el algoritmo, sino por una mala definición del problema.

ThinkML estructura el razonamiento previo al modelado.

## Qué hace
- Clarifica el tipo de problema de datos
- Sugiere estrategias de modelado y métricas
- Señala riesgos antes de implementar
- Propone próximos pasos concretos

## Qué NO es
- No carga datasets
- No entrena modelos
- No es AutoML
- No reemplaza al análisis

Esta herramienta organiza el pensamiento, no ejecuta el modelo.
